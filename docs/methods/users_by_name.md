### /users/name/:name 
Coming soon
{: .label .label-yellow }

Получает игрока по его игровому никнейму

Параметры: 

Параметр | Тип
------------ | -------------
`name` | string

Возвращает информацию об игроке по его игровому никнейму. Если игрока с таким никнеймом нет, в выводе будет пустой массив.
